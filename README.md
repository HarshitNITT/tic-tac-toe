# tic-tac-toe
This is a web-app which can used for making,storing and deleting notes in react.

# Installation
First clone the project in your local machine using:
~~~
git clone https://github.com/HarshitNITT/tic-tac-toe
npm i
cd tic-tac-toe
npm start
~~~
